# VIM CONFIG

## Instructions

> you should have installed vim and vundle  

```
sudo apt-get install vim
git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
```
> update config files  

```
cp ./vimrc ~/.vimrc
tar -zxvf vim_plug.tgz
cp -r .vim/* ~/.vim/
```
